# מערכת הויקי - יקומות

## סקירה כללית

מערכת ויקי מלאה לניהול תוכן על יקומות, הכוללת:
- **דמויות** - כל הדמויות ביקום
- **תכניות** - תכניות טלוויזיה/רדיו
- **פרסומות** - פרסומות מהיקום
- **מושגים** - מושגים מרכזיים

## תכונות עיקריות

### 1. דף בית מדהים
- אפקטים ויזואליים (כוכבים, גרדיאנטים, parallax)
- סטטיסטיקות לכל קטגוריה
- עיצוב מודרני ומרשים

### 2. דפי רשימה
- גריד תמונות יפה
- סטטיסטיקות (צפיות, אימות)
- חיפוש וסינון

### 3. דפי פרטים (כמו ויקיפדיה)
- תוכן עשיר
- תמונות
- קישורים חיצוניים
- קישוריות לרפרנסים בפרקים

### 4. מערכת עריכה
- עריכה ישירה בדף
- היסטוריית שינויים
- מערכת ניקוד

### 5. מערכת ניקוד
- יצירת ערך חדש: 10 נקודות
- עריכת ערך: 2 נקודות
- אימות ערך: נקודות נוספות

### 6. מניעת ספאם
- מערכת אימות עריכות
- דרישת התחברות לעריכה
- מעקב אחר שינויים

## הגדרה

### שלב 1: הרצת סקריפטים SQL

1. **setup-chapters-system.sql** - מערכת הפרקים
2. **setup-wiki-system.sql** - מערכת הויקי (חדש!)

### שלב 2: מבנה מסד הנתונים

#### טבלאות עיקריות:
- `characters` - דמויות
- `programs` - תכניות
- `advertisements` - פרסומות
- `concepts` - מושגים

#### טבלאות תמיכה:
- `reference_connections` - קישורים בין רפרנסים לערכים
- `edit_history` - היסטוריית עריכות
- `edit_approvals` - אישורי עריכות

## שימוש

### הוספת ערך חדש
1. התחבר למערכת
2. עבור לקטגוריה הרצויה (דמויות/תכניות/וכו')
3. לחץ על "הוסף חדש"
4. מלא את הפרטים ושמור

### עריכת ערך
1. פתח את דף הערך
2. לחץ על "ערוך"
3. ערוך את התוכן
4. שמור (תקבל 2 נקודות)

### קישור לרפרנס
1. פתח את דף הערך
2. לחץ על "קשר רפרנס"
3. בחר רפרנס מהפרקים
4. הרפרנס יופיע בדף הערך

## נקודות

- **יצירת ערך**: 10 נקודות
- **עריכת ערך**: 2 נקודות
- **אימות ערך**: נקודות נוספות (לפי מערכת המוניטין)

## מניעת ספאם

1. **דרישת התחברות** - רק משתמשים מחוברים יכולים לערוך
2. **היסטוריית שינויים** - כל עריכה נשמרת
3. **מערכת אימות** - משתמשים יכולים לאמת עריכות
4. **מעקב נקודות** - משתמשים עם נקודות נמוכות מוגבלים

## עיצוב

- עיצוב מודרני עם אפקטים ויזואליים
- תמיכה מלאה ב-RTL ועברית
- אנימציות חלקות
- רספונסיבי מלא

## דפים

- `/` - דף הבית
- `/characters` - רשימת דמויות
- `/characters/[id]` - דף דמות
- `/characters/new` - הוספת דמות חדשה
- `/programs` - רשימת תכניות
- `/programs/[id]` - דף תכנית
- `/advertisements` - רשימת פרסומות
- `/advertisements/[id]` - דף פרסומת
- `/concepts` - רשימת מושגים
- `/concepts/[id]` - דף מושג

## הערות

- כל הערכים ניתנים לעריכה על ידי כל משתמש מחובר
- מערכת הניקוד מעודדת תרומה איכותית
- הקישוריות לרפרנסים מאפשרת ניווט קל בין תוכן
